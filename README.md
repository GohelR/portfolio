# Ravi Gohel Portfolio

A modern premium personal portfolio website for an AI & Machine Learning Engineer built with **React + Tailwind CSS**.

## Tech Stack
- React 18
- Vite 5
- Tailwind CSS 3

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Update links
All primary output links (resume, GitHub, LinkedIn, project GitHub/demo URLs) are centralized in `src/App.jsx` data objects for easy editing.
